# 图标资源说明

## TabBar 图标

需要准备以下 TabBar 图标（建议尺寸：81x81px）：

- `home.png`: 首页图标（未选中）
- `home-active.png`: 首页图标（选中）
- `stats.png`: 统计图标（未选中）
- `stats-active.png`: 统计图标（选中）
- `bills.png`: 账单图标（未选中）
- `bills-active.png`: 账单图标（选中）
- `profile.png`: 我的图标（未选中）
- `profile-active.png`: 我的图标（选中）

## 默认头像

- `default-avatar.png`: 默认用户头像（建议尺寸：200x200px）

## 图标风格

所有图标应遵循以下规范：

1. **线条风格**: 使用线性图标，线条粗细一致
2. **颜色规范**:
   - 未选中: `#86868b`（灰色）
   - 选中: `#0071e3`（Apple 蓝）
3. **设计原则**: 简洁、现代、易识别
4. **圆角处理**: 适当的圆角，增加亲和力

## 图标来源

可以从以下平台获取图标：

1. Iconfont: https://www.iconfont.cn/
2. SF Symbols（需 Mac）
3. Feather Icons: https://feathericons.com/
4. Heroicons: https://heroicons.com/

## 自定义图标

如需自定义图标，建议使用设计工具：

- Sketch（Mac）
- Figma
- Adobe XD
- Illustrator

## 注意事项

1. 确保图标清晰度，避免模糊
2. 控制图标文件大小，建议每个图标不超过 10KB
3. 使用 PNG 格式，支持透明背景
4. 保持图标风格统一
